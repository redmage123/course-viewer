Generative AI Fundamentals

AI That Creates New Content

What is Generative AI?

AI that can create new, original content based on patterns learned from training data.

  • Text generation
  • Image generation
  • Code generation
  • Audio/music generation

Large Language Models (LLMs)

  • Trained on massive text datasets
  • Predict next tokens in sequence
  • Understand context and nuance
  • Examples: GPT-4, Claude, Llama

Key Concepts

Tokens

Units of text (words or word parts) that LLMs process

Prompts

Input text that guides the model's response

Completions

Generated output from the model

Temperature

Controls randomness/creativity (0=deterministic, 1=creative)

Generative AI creates NEW content; it doesn't just analyze or classify existing content like traditional AI.

Prompt Engineering

Crafting Effective AI Instructions

Prompt Components

  • System Message: Sets behavior and context
  • User Message: The actual request
  • Assistant Message: Model's response
  • Examples: Few-shot learning samples

Best Practices

  • Be specific and clear
  • Provide context
  • Use examples (few-shot)
  • Break complex tasks into steps
  • Specify output format

Prompt Techniques

Technique Description
Zero-shot No examples provided
Few-shot Include example inputs/outputs
Chain of Thought Ask model to explain reasoning
Role Playing Assign persona to model
System messages persist throughout conversations and establish model behavior. User messages are individual requests.

What is RAG?

A pattern that combines:

  1. Retrieval: Find relevant documents
  2. Augmentation: Add to prompt context
  3. Generation: LLM creates response

Benefits

  • Access current information
  • Cite sources
  • Reduce hallucinations
  • Domain-specific responses

RAG Architecture

1
Index Data - Store in Azure AI Search
2
Embed Query - Convert to vectors
3
Search - Find relevant chunks
4
Augment - Add to system prompt
5
Generate - LLM responds with context
RAG = Retrieval + Augmentation + Generation. It uses Azure AI Search with Azure OpenAI to ground responses in your data.
