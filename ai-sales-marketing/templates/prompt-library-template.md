## Email Marketing Prompts

### Welcome Email
```
Role: You are an email marketing specialist focused on B2B onboarding.

Context: A new user just signed up for [PRODUCT]. They are a [JOB_TITLE] at a [COMPANY_SIZE] company in the [INDUSTRY] industry.

Task: Write a welcome email that:
- Thanks them for signing up
- Highlights 3 key features to explore first
- Includes a clear next step

Format:
- Subject line (under 50 characters)
- Preheader text (under 100 characters)
- Body (under 200 words)
- Single CTA button text

Tone: Friendly, helpful, not salesy.
```

### Follow-Up Email (Post-Demo)
```
Role: You are an experienced B2B sales development rep.

Context: [PROSPECT_NAME] from [COMPANY] had a demo yesterday. Key interests: [INTERESTS]. Main concern: [OBJECTION].

Task: Write a follow-up email that:
- References something specific from the call
- Addresses their main concern
- Provides one relevant resource
- Proposes clear next steps

Format: Subject line + 150-word email max.

Tone: Professional, consultative, confident but not pushy.
```

### Re-Engagement Email
```
Role: You are a customer success specialist.

Context: [CUSTOMER_NAME] was an active user of [PRODUCT] but hasn't logged in for 30 days. They previously used features: [FEATURES].

Task: Write a re-engagement email that:
- Acknowledges it's been a while
- Highlights what's new since they left
- Offers help or a quick call
- Creates gentle urgency

Format: Subject line + 120-word email.

Tone: Caring, not guilt-tripping. Helpful, not desperate.
```

## Social Media Prompts

### LinkedIn Thought Leadership Post
```
Role: You are a [JOB_TITLE] with expertise in [TOPIC].

Context: You want to share insights about [SPECIFIC_TOPIC] with your professional network.

Task: Write a LinkedIn post that:
- Opens with a hook (question, bold statement, or surprising stat)
- Shares 3-5 actionable insights
- Ends with a question to drive engagement

Format:
- 200-300 words
- Use line breaks for readability
- Include 3-5 relevant hashtags
- End with a question

Tone: Authoritative but approachable. Share genuine insights, not promotional content.
```

### Twitter/X Thread
```
Role: You are a marketing thought leader.

Context: Topic is [TOPIC]. Audience is [TARGET_AUDIENCE].

Task: Create a 5-tweet thread that:
- Hook tweet grabs attention
- Tweets 2-4 provide value/insights
- Final tweet has a clear CTA

Format:
- Each tweet under 280 characters
- Use numbers for each tweet (1/5, 2/5, etc.)
- Include 1-2 relevant hashtags on final tweet only

Tone: Conversational, punchy, valuable.
```

### Instagram Caption
```
Role: You are a creative social media manager.

Context: We're posting about [TOPIC]. Image shows [IMAGE_DESCRIPTION].

Task: Write an Instagram caption that:
- Opens with attention-grabbing first line
- Tells a brief story or shares insight
- Ends with a call-to-action

Format:
- 100-150 words
- Include line breaks
- 3-5 relevant emojis
- 10-15 hashtags (listed after main caption)

Tone: Engaging, authentic, slightly playful.
```

## Content Marketing Prompts

### Blog Post Outline
```
Role: You are an SEO-focused content strategist.

Context: Target keyword: [KEYWORD]. Audience: [AUDIENCE]. Goal: [GOAL].

Task: Create a comprehensive blog post outline that:
- Has an attention-grabbing H1 title (60 chars max)
- Includes 5-7 H2 sections with brief descriptions
- Suggests 2-3 H3 subsections where appropriate
- Identifies spots for examples/case studies
- Includes meta description (155 chars max)

Format: Hierarchical outline with title, meta, and section headers.

Note: Focus on search intent and providing genuine value.
```

### Case Study Summary
```
Role: You are a B2B content writer specializing in case studies.

Context:
- Customer: [CUSTOMER_NAME] in [INDUSTRY]
- Challenge: [CHALLENGE]
- Solution: [HOW_WE_HELPED]
- Results: [KEY_METRICS]

Task: Write a case study summary that:
- Opens with the impressive result
- Briefly describes the challenge
- Explains the solution
- Highlights 3 key results with numbers

Format: 250-300 words. Include a compelling headline and one customer quote placeholder.

Tone: Professional, results-focused, credible.
```

### Product Description
```
Role: You are a conversion copywriter.

Context: Product: [PRODUCT]. Target buyer: [PERSONA]. Main benefit: [BENEFIT]. Key features: [FEATURES].

Task: Write a product description that:
- Leads with the primary benefit
- Describes how it solves the customer's problem
- Highlights 3-4 key features
- Ends with clear CTA

Format: 150-200 words for main description + 3-4 bullet points for features.

Tone: Clear, benefit-focused, persuasive but not hypey.
```

## Advertising Prompts

```
Role: You are a PPC advertising specialist.

Context:
- Product/Service: [PRODUCT]
- Target audience: [AUDIENCE]
- Key benefit: [BENEFIT]
- Competitor advantage: [DIFFERENTIATOR]

Task: Generate 5 Google Ads variations with:
- Headline 1 (30 chars max)
- Headline 2 (30 chars max)

```

```

- Target audience: [AUDIENCE]
```

## Sales Support Prompts

### Prospect Research
```
Role: You are a sales research analyst.

Context: I'm preparing for a call with [CONTACT_NAME], [TITLE] at [COMPANY].

Task: Based on publicly available information, provide:
1. Company overview (size, industry, recent news)
2. Likely pain points based on their role and company
3. Potential objections to anticipate
4. Conversation starters/rapport builders
5. Questions to ask during discovery

Format: Bullet points organized by category.

Note: Only use publicly available information.
```

### Objection Handling
```
Role: You are a sales coach with 15 years of B2B SaaS experience.

Context: Product: [PRODUCT]. Price: [PRICE]. Objection: "[OBJECTION]"

Task: Provide 3 different ways to handle this objection:
1. Acknowledge and redirect approach
2. Question-based approach (dig deeper)
3. Social proof approach

For each, include:
- What to say (word-for-word)
- Why this works
- Follow-up question to ask

Tone: Consultative, empathetic, never aggressive.
```

### Meeting Summary Email
```
Role: You are an organized sales professional.

Context:
- Meeting with: [ATTENDEES]
- Topics discussed: [TOPICS]
- Key decisions: [DECISIONS]
- Next steps: [NEXT_STEPS]

Task: Write a meeting summary email that:
- Thanks attendees for their time
- Recaps key discussion points (3-5 bullets)
- Lists agreed-upon next steps with owners
- Confirms next meeting/follow-up

Format: Professional email under 200 words.

Tone: Organized, action-oriented, appreciative.
```

## Analytics Prompts

### Campaign Analysis
```
Role: You are a marketing analytics expert.

Context: Campaign data:
- Campaign: [CAMPAIGN_NAME]
- Goal: [GOAL]
- Metrics: [PASTE_METRICS]
- Industry benchmarks: [BENCHMARKS]

Task: Analyze this campaign and provide:
1. Performance summary (what worked, what didn't)
2. 3 key insights from the data
3. Root cause analysis for underperformance
4. 5 specific recommendations for improvement
5. A/B tests to run next

Format: Structured analysis with clear headers.

Tone: Data-driven, actionable, strategic.
```

### Report Summary
```
Role: You are a marketing director preparing for an executive briefing.

Context: [PASTE_DATA_OR_REPORT]

Task: Create an executive summary that:
- Leads with the headline (best or most important finding)
- Summarizes key metrics in 3-5 bullets
- Highlights wins and challenges
- Recommends next steps

Format: One page max. Use bullet points. Include specific numbers.

Tone: Executive-level, concise, strategic.
```

## Iteration Phrases

Use these to refine AI outputs:

**To make it shorter:**
- "Shorten this to under [X] words while keeping key points."
- "Give me a more concise version."

**To make it longer:**
- "Expand the section on [topic] with more detail."
- "Add more examples and context."

**To change tone:**
- "Make this more [formal/casual/urgent/friendly]."
- "Rewrite in a more conversational tone."

**To improve quality:**
- "Make this more specific and actionable."
- "Add concrete examples."
- "Challenge this analysis. What am I missing?"
- "Give me 3 alternative approaches."

**To fix issues:**
- "This sounds too salesy. Make it more helpful."
- "Remove jargon and simplify the language."
- "Make the opening hook stronger."

## How to Use This Library

1. **Copy the template** you need
2. **Fill in the [VARIABLES]** with your specific information
3. **Paste into ChatGPT, Claude, or your preferred AI tool**
4. **Review and refine** the output
5. **Save successful prompts** with notes on what worked

**Pro Tip:** Create a document with your most-used prompts pre-filled with your company's standard information (product names, value props, personas) so you can quickly generate content.
